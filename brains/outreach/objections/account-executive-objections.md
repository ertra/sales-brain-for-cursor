# Account Executive - Objection Handling

## Overview
AE objections are usually practical: extra clicks, workflow interruption, and concern that automation harms personalization.

## Objection Plays

### 1. Workflow burden: "This will create more admin for me"
- **Why They Say This**: Prior tools increased data-entry work.
- **Acknowledge**: "No rep wants another admin tool."
- **Respond**: "We'll focus on automation that removes post-call and follow-up burden first."
- **Proof Point**: Meet/Call include automated summary/action-item support; Deal provides activity context.
- **Follow-Up Question**: "Which admin task would you most want eliminated first?"

### 2. Price/Budget: "I'd rather spend on more pipeline"
- **Why They Say This**: Reps value direct pipeline generation spend.
- **Acknowledge**: "Makes sense to prioritize pipeline impact."
- **Respond**: "This should be judged by conversion and cycle-time improvements from your existing pipeline."
- **Proof Point**: Customer listings show meeting-to-opportunity and qualified opportunity gains.
- **Follow-Up Question**: "Would faster conversion on existing pipeline be as valuable as more top-funnel volume?"

### 3. Timing: "I can't change my process mid-deal cycle"
- **Why They Say This**: Fear of disrupting active opportunities.
- **Acknowledge**: "Protecting active deals is the right priority."
- **Respond**: "We'll apply changes to a controlled segment first, not every open opportunity."
- **Proof Point**: Modular deployment by package/use case.
- **Follow-Up Question**: "Which segment is safest for a controlled pilot?"

## Operator Guidance
### Questions to Ask Early
1. Which tasks consume too much AE time weekly?
2. Where do deals stall most often?
3. How are follow-up and next steps managed today?
4. Which KPI should improve first: cycle time, stage conversion, or win rate?

### Disqualification Criteria
- AE team rejects any workflow standardization.
- No baseline metrics for conversion/cycle-time.

### Talk Tracks
- **15s**: This is about reducing AE admin and improving deal progression, not adding process.
- **30s**: We start with one workflow where automation saves time and increases conversion quality.
- **2min**: AE resistance usually comes from tool fatigue. By targeting one high-friction activity first and proving impact on conversion or cycle-time, adoption becomes outcome-driven rather than compliance-driven.

### Next-Step CTA
- Run a 30-day AE pilot on one stage bottleneck.
