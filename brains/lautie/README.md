# LAUTIE (Lautie EDC)

**Stay Cool & Be Different**

## About

LAUTIE EDC is the premier metal fidget toy brand, created by craftsman Yangman and designer Niko. They specialize in original everyday carry (EDC) products for fidget enthusiasts: fidget spinners, sliders, haptic coins, rings, pendants, and related accessories. The brand emphasizes premium quality, in-house manufacturing (“Lautie factory”), and a distinct identity. This folder contains the **Sales Brain** for LAUTIE: structured sales and product context used for AI-assisted research, messaging, and outreach.
